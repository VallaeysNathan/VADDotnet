# Dotnet VAD Servie experiment (for Linux)

This is a small dotnet console application (for linux) where a VAD model is used to detect when a person has started speaking.
This was developed as a test to be used in a non explicit turn based Voice Agent.

Model and starting point of the code came from this github repo: https://github.com/snakers4/silero-vad/tree/master/examples/csharp
the linked github repo is under the MIT license as is this one.
